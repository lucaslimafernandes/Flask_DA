Flask, Docker, Ansible

A simple (very simple) application to learn to use Docker and Ansible to future uses from my repositories.

## How to

Install ansible:

    pip install ansible

Verify:

    pip show ansible

Running:

    ansible-playbook --ask-become-pass playbook.yml

Finally:

    http://localhost:5099/
